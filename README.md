## ⚡️ Requirements

- eslint >= **^7.32.0 || ^8.2.0**

## 📦 Installation

Run this command to install eslint and the plugin.

```bash
pnpm i -D eslint-config-coder-dev
```

## 🚀 Usage

To use this plugin put in your eslint config file this lines of codes.

```json
{
  "root": true,
  "extends": "coder-dev"
}
```
